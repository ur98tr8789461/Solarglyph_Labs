Workflow

                
Simple Local Installation

                

                    Since Developer Intelligence operates as a fully localized Roblox tool, installation is straightforward and security-focused. No external injectors, no complicated setups.
                

            


            
            

                
                
                

                    

                    

                        

                            1
                        

                        
Retrieve the File

                        

                            Purchase and download the official plugin assets directly via our secure Parcel Roblox storefront. You will receive a high-performance local plugin workspace file (`.rbxm` or source directory).
                        

                    

                    

                        
                    

                


                
                

                    

                    

                        

                            2
                        

                        
Import & Configure

                        

                            Drag and drop the downloaded asset file into your active Roblox Studio project. Navigate inside the package, open the Configuration script module, and insert your unique API credential token.
                        

                    

                    

                        Roblox Studio integration
                    

                


                
                

                    

                    

                        

                            3
                        

                        
Save to Local Plugins

                        

                            Right-click the primary model folder inside your Studio explorer tree and select the "Save as Local Plugin" option from the context menu. You're done! Your toolbar is now loaded with AI power.
